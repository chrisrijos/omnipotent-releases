Cascade Architecture

Three tiers. Zero cloud calls.

Fast rules handle the obvious threats. A 1B ONNX classifier catches the subtle ones. A full 3B LLM runs only for ambiguous edge cases. Each tier runs locally — escalating only when confidence is low. 90% of requests resolve in the first tier at sub-millisecond latency.

Tier 1: Heuristic

< 1ms · Regex + Entropy + Rules

50+ deterministic patterns: injection signatures, PII formats (SSN, MRN, credit card), API key patterns, and entropy-based secret detection. Zero compute cost.

90%of threats stopped here

Tier 2: ONNX Classifier

~35ms · 1B Parameter Model

Lightweight neural classifier for semantic analysis. Catches obfuscated injections, context-aware PII, and encoded payloads that bypass regex patterns. ONNX runtime, no GPU required.

~8%escalated from Tier 1

Streaming + Veto (Buffered Release)

Tool output is buffered briefly while the security scan runs in parallel. If the verdict arrives before the buffer window expires, output is released immediately (PASS) or discarded entirely (BLOCK). Sensitive content never reaches the client.

2. Parallel

run simultaneously

3. Buffer

Tool output held
up to Tmax

4. Release

PASS → release
