FAQ

Common Questions

Answers for developers evaluating Omnipotent.

What is Omnipotent?

Omnipotent is a Federated AI Sovereign Operating System — a multi-agent orchestration platform that runs entirely on user-owned infrastructure with no external SaaS dependencies. It coordinates 12 specialized AI agent personas through two closed-loop orchestration protocols with mandatory human checkpoints, a Signal Bus architecture, and tiered memory (Hot/Cold/Federated).

How do I install Omnipotent?

Install via the Rust CLI (cargo install --path . then omnipotent init) or Docker (docker-compose -f docker-compose.omnipotent.yml up). Pre-built binaries for macOS and Linux are coming soon. See the GitHub repository for full quickstart instructions.

Is Omnipotent free?

Omnipotent is currently in active development. Source code is available for authorized users on GitHub. The project is designed around data sovereignty — your hardware, your data, your agents — with no recurring SaaS fees or cloud dependencies.

What agents does it include?

12 specialized personas form The Legion: Archon (Kernel Architect), Oracle (Intel and Research), Shadow Route (Ops and Orchestration), Validitus (Security and QA), Codex (Ancient Wise), Eidolon (UI/Voice Creative), Artificer (CI/CD and Self-Upgrade), Diplomat (Federation Sync) — plus sub-roles Quartermaster, Scout, Mnemon, and Mercator.
