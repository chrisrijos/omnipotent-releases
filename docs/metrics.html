Adding Competitor Benchmarks

We do not publish invented competitor numbers. To add competitors, append entries to public/validation/competitors.json (or use the helper script in aiegis-module-rs/scripts/).

[
  {
    "system": "Competitor Name",
    "profile": "label",
    "unit": "us",
    "min": 0,
    "p50": 0,
    "p95": 0,
    "max": 0,
    "platform": { "os": "linux", "arch": "x86_64" },
    "notes": "Hardware + config notes",
    "source": "Link or internal run ID"
  }
]
