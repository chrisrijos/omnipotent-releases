Integration

Three ways to integrate.

Proxy mode for zero-config. Sidecar for infrastructure teams. SDK for embedded classification. Pick what fits your stack.

Proxy Mode

default

Point your AI SDK at localhost:8080. Zero code changes.

  • Drop-in replacement for any OpenAI-compatible API
  • Transparent TLS interception with auto-generated CA
  • Works with LangChain, Vercel AI SDK, OpenAI SDK
  • JSON structured logging for every request

Sidecar Mode

infrastructure

Deploy alongside your service. Docker, K8s, or bare metal.

  • Official Docker image and Helm chart
  • Health checks and Prometheus metrics endpoint
  • Configurable resource limits and scaling
  • Shared TOML config with hot-reload support

SDK Mode

embedded

Import the classifier directly. TypeScript, Python, Rust.

  • Direct FFI to the local LLM classifier
  • Async/await API with typed responses
  • Batch classification for pipeline workloads
  • Custom rule registration at runtime
